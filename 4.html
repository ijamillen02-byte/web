Fomulários

O HTML possui a tag form que possibilita definir fomulários para submissão de dados informados pelo usuário por meio de campos de input, dentro do formulário. A tag <form> possui além dos atributos globais e também atributo action="url", utilizado para definir o destino dos dados submetidos. Exemplo: 

 
            <form action="url">
            <input type="text" name="variavelA"
            ...
            </form>

Table

A tag table é utilizada par adefinição de tabela no HTML, internamente em uma tabela podemos criar linhas "rows" utilizando a tag <tr> e celular dentro das linhas utilizando a tag <td> exemplo: 

        <table> 
            <tr> 
                <td> Celula 1x1 </td> <td> Celula 1x2 </td>
              </tr>
              <tr> 
                <td> Celula 2x1 </td> <td> Celula 2x2 </td>
              </tr>
         </table>
O código anterior define a tabela abaixo
Celula 1x1 Celula 1x2
Celula 2x1 Celula 2x2
Carnes Id item churrasco valor
1 carne 5kg R$200
2 frango 2kg R$40
Bebida 4 Refrigerante 2L R$20
