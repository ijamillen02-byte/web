O que é HTML

São nottações utilizas para descrever aspectos qualquer de documentos por meio de tags.

A estrutura do HTML se resume em:

DOCTYPE> é a declaração do documento. que é usada para especificar ao navegador o tipo de página html que ele vai redenrizar.


HTML É usada para dizer ao navegador onde iniciar a interpretação.


Head é o cabeçalho, onde as são colocados as informações que serão usadas pelo navegador, como título, link e etc.


Body marca o início do corpo do documento, todo documento dados dele devem estar dentro do body.


Ordem do arquivo HTML:

  • DOCTYPE>
  • HTML
  • head
  • fechamendo to head
  • body
  • fechamento do body
  • fechamento do HTML
