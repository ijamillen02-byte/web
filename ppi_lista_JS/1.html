O que é JavaScript?

A. O sites possuiam documentos html estáticos, com pouca ou até sem imagens, essa sendo a razão e inspiração para os pesquisadores buscarem algo que os usuários pudessem interagir.


B. Enquando a execução no navegador é feita no computador e dentro de um ambiente do navegador, a no Servidor, o código é processado no servidor web remoto.


C. É um mecanismo de segurança que isola os processos de execução do navegador, do sistema operacional do computador.


D. Eles são propriedades do window:

Document: representa o Document Object Model, e fornece métodos e propriedades para acessar e manipular a estrutura, o conteúdo e seu estilo.

Console: fornece acesso a um interface de depuração do navegador, oferecendo métodos de registrar informações, erros, avisos e outros dados.

console.warn("Este é um aviso"); console.error("Este é um erro");

Alert: exibe uma caixa de diálogo com uma mensagem e um botão OK.

Exibir um alerta alert("Seja bem vindo!);
