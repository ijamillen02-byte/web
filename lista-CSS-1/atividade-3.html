Como podemos incluir CSS em documentos HTML?

Há 3 formas:

  1. CSS inline;
  2. CSS interno;
  3. CSS externo;

CSS inline

Ele aplica estilos diretamente a um elemento HTML usando o atributo <style>. É útil para estilos específicos de um único elemento, mas não recomendado para grandes projetos. Exemplo: 

<p style="color: blue"> Esse texto ficará azul.</p>

CSS interno

A tag <style> para inserir códigos CSS dentro da seção <head> do seu documento html. Ele é útil para estilos que se aplicam apenas a uma página específica. Exemplo: 

<p style> para inserir códigos CSS dentro da seção < head>
<style>
h1{
    color: red;
}
</style>
</head>
</body>
< h1> este título é vermelho.</h1

</body>

CSS externo

Este é o método masi comum e recomendado, especialmente para projetos maiores. Ele envolve a criação de um arquivo9 .css separado e a vinculação dele ao seu HTML usando a tag <link> na seção <head>. Exemplo:

No arquivo HTML:
            <head> 
                <link rel="stylesheet" href"style.css">
            </head>
No arquivo CSS (style.css):
                h1{
                    color: yellow;
                }
