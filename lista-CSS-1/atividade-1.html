O que é uma regra CSS? Explique o conceito de regra CSS e forneça um exemplo prático que inclua um seletor, uma propriedade e um valor.

É uma ferramenta com recursos para estilizar e posisionar elementos em página html. Há a declaração e o seletor, onde a declaração vai definiir a aparência como as cores, tamanhos, linhas e afins, enquanto o seletor seria os índices para poder incluir as declarações dentro deles, no caso vai definir qual elemento a regra será aplicada.
